int fib(int n){
  if(n <= 1)
    return n;
  return fib(n-1) + fib(n-2);
}

void bubbleSort(int a[], int n){
  int i,j;
  bool swapped;
  for(i=0;i < n-1;i++){
    swapped = false;
    for(j=0;j < n-i-1; j++){
    if(a[j] > a[j+1]){
      swap(&a[j],&a[j+1]);
      swapped = true;
    }
  }
  if(swapped == false){
    break;
  }
}}

Palindrome

A word, phrase, or sequence that reads the same backward as forward

Fibonacci

A series of numbers in which each number ( Fibonacci number ) is the sum of the two preceding numbers. The simplest is the series 1, 1, 2, 3, 5, 8, etc.

Factorial of a Number

The product of an integer and all the integers below it; e.g. factorial four ( 4! ) is equal to 24.

Bubble Sorting

Bubble sort, sometimes referred to as sinking sort, is a simple Sorting algorithmthat repeatedly steps through the list to be sorted, compares each pair of adjacent items and swaps them if they are in the wrong order.
